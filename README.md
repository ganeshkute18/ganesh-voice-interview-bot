# Ganesh Voice Interview Bot

A voice-based interview conducting application that leverages AI technology to facilitate automated interviews. The Ganesh Voice Interview Bot includes features for resume parsing and AI-powered conversations, making the interview process more efficient and user-friendly.

## Features

- **Voice-Based Interviews**: Conduct interviews using voice commands for a more interactive experience.
- **Resume Parsing**: Upload and process resumes to extract relevant information automatically.
- **AI-Powered Responses**: Utilize the Groq API for generating intelligent responses during interviews.
- **Customizable Interface**: A redesigned interview setup interface that can be tailored to suit different needs.
- **Multi-Branch Development**: Explore various branches (10 available) for different features and improvements.

## Technology Stack

- **Backend**: Python (36.0%)
- **Frontend**: HTML (64.0%)
- **AI Integration**: Groq API
- **Web Framework**: Utilizes a web framework for application structure

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/ganeshkute18/ganesh-voice-interview-bot.git
   cd ganesh-voice-interview-bot
   ```

2. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   python app.py
   ```

## Usage

- Upload your resume through the interface.
- Start the voice interview process by following the prompts.
- Interact with the bot using voice commands to answer questions.
